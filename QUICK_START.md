# 🦄 Quick Start Guide - Unicorn Supea

## 🚀 Get Started in 3 Steps

1. **Start the website:**
   ```bash
   npm run dev
   ```
   Then open http://localhost:4321

2. **Add new art (easy way):**
   ```bash
   npm run add-art
   ```
   Follow the prompts!

3. **Add new art (manual way):**
   - Put image in `public/art/`
   - Edit `src/data/art.ts`
